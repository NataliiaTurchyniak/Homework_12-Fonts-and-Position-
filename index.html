CORE COMPONENTS

React Native is an open source framework for building Android and iOS applications using React and the app platforms native capabiles. With React Native, you use JavaScript to access your platform's APIs as well as to describe the appearance and behavior of your UI using React components: bundles of reusable, nestable code. You can learn more about React in the text section. But first,let's cover how components work in React Native.

React Native runs on React, a popular open source library for building user interfaces with JavaScript. To make the most of React Native, it helps to understand React itself.
